

Link to Laravel Websocket installation: 
https://beyondco.de/docs/laravel-websockets/getting-started/introduction

commands: 
------------
composer require beyondcode/laravel-websockets

php artisan vendor:publish --provider="BeyondCode\LaravelWebSockets\WebSocketsServiceProvider" --tag="migrations"

php artisan migrate

php artisan vendor:publish --provider="BeyondCode\LaravelWebSockets\WebSocketsServiceProvider" --tag="config"


Pusher replacement: 
(Install a lower version of pusher)
https://beyondco.de/docs/laravel-websockets/basic-usage/pusher

composer require pusher/pusher-php-server "~5.0"  (gumana to sa windows ko, paki-experiment na lang kung anong version ang gagana sa AWS)


Composer Installation commands: 
====================================

composer update


NPM commands: 
====================================

npm install 

npm install laravel-echo pusher-js

npm run dev (to compile code to public resources



To run the websocket service: 
====================================

php artisan websocket:server


to run the PHP web server: 
====================================
php artisan serve
